Register.php, где пользователь может делать регистрацию. 

Login.php, где пользователь авторизуется на сайт. 

Main.php - главная страница, которая открывается после авторизации. 
Здесь можно увидеть общие контакты, доступные для всех пользователей. 

Рядом с каждым контактом есть кнопка «добавить», которая добавляет контакты в Избранное. 
У каждого пользователя есть свой список избранных контактов. 
После нажатия кнопки или навигационной ссылки «Избранное», 
которая находится на левом углу главной страницы открывается страница с вашими избранными контактами. 
_________________________________________________________________________________________________
Signup.php, signin.php, add.php в этих страницах используется метод пост, 
чтобы зарегистрировать или авторизовать пользователя и сохранить избранные  контакты. 
_________________________________________________________________________________________________
Connect.php используется для соединения базы данных с html страницей. 

Logout.php используется для выхода из сессии. 
_________________________________________________________________________________________________
В папке "Скриншоты" можно увидеть скриншоты страниц и базы данных. 
_________________________________________________________________________________________________
Для проекта были использованы Visual Code, как текстовый редактор и MAMP для базы данных.
